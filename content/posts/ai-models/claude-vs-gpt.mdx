## 能力对比

### 1. 代码能力

**GPT-4 的优势：**
- 代码补全更准确
- 调试能力更强
- 支持更多编程语言

<PromptBlock content={`# 用 GPT-4 写一个快速排序算法
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)`} title="代码示例" />

## 场景推荐

### 适合使用 Claude 的场景

1. **长文档处理**：处理书籍、论文、长报告
2. **代码审查**：审查代码质量和安全性
3. **创意写作**：小说、剧本、诗歌
4. **数据分析**：解释数据、生成报告

### 适合使用 GPT-4 的场景

1. **代码开发**：写代码、调试、重构
2. **多模态任务**：图片理解、图表生成
3. **API 集成**：开发者工具集成
4. **复杂推理**：数学问题、逻辑推理
