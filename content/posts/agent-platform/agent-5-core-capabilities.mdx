# 智能体5大核心能力落地指南

AI Agent（智能体）是2024-2025年最火热的AI技术方向。但市面上大多数"智能体"只是简单的Prompt包装，离真正的自主智能体还有很大差距。

本文将从**技术原理 + 代码实现**两个维度，深入讲解如何构建具备真正自主能力的AI Agent。

## 一、自主感知：Agent的眼睛和耳朵

### 什么是自主感知？

自主感知是指Agent能够**主动获取外部信息**的能力，不仅是响应用户输入，而是能够主动查询环境状态。

### 核心实现方式

**1. 工具调用（Tool Use）**

## 二、自主记忆：Agent的大脑

### 什么是自主记忆？

人类之所以智能，是因为能**记住过去的经验**。Agent同样需要记忆系统，否则每次对话都是"零认知起点"。

### 三层记忆架构

```
┌─────────────────────────────────────────┐
│           长期记忆 (Long-term)          │
│  ┌─────────────────────────────────┐   │
│  │ 知识库 / RAG / 向量数据库       │   │
│  │ 文档、规则、用户偏好            │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
                    ▲
                    │ 定期沉淀
                    │
┌─────────────────────────────────────────┐
│           中期记忆 (Mid-term)           │
│  ┌─────────────────────────────────┐   │
│  │ 会话摘要 / 关键事件记录          │   │
│  │ 当前任务进度                    │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
                    ▲
                    │ 信息过滤
                    │
┌─────────────────────────────────────────┐
│           短期记忆 (Short-term)         │
│  ┌─────────────────────────────────┐   │
│  │ 当前对话上下文                  │   │

```

### 代码实现：向量数据库记忆

```python
from openai import OpenAI
import chromadb
import json

client = OpenAI()

# 初始化向量数据库
db = chromadb.PersistentClient(path="./memory_db")
collection = db.get_or_create_collection("agent_memory")

def store_memory(content: str, metadata: dict):
    """存储记忆到向量数据库"""
    embedding = client.embeddings.create(
        input=content,
        model="text-embedding-3-small"
    ).data[0].embedding
    
    collection.add(
        embeddings=[embedding],
        documents=[content],
        metadatas=[metadata],
        ids=[f"mem_{metadata['timestamp']}"]
    )

def retrieve_memory(query: str, top_k: int = 5) -> list:
    """检索相关记忆"""
    query_embedding = client.embeddings.create(
        input=query,
        model="text-embedding-3-small"
    ).data[0].embedding
    
    results = collection.query(
        query_embeddings=[query_embedding],
        n_results=top_k
    )
    return results['documents'][0]

def chat_with_memory(user_input: str) -> str:
    """带记忆的对话"""
    # 1. 检索相关记忆
    relevant_memories = retrieve_memory(user_input)
    
    # 2. 构建带记忆的上下文
    context = "\n\n".join([
        f"[相关记忆] {mem}" for mem in relevant_memories
    ])
    
    messages = [
        {"role": "system", "content": f"你是一个智能助手。以下是用户的背景信息：\n{context}"},
        {"role": "user", "content": user_input}
    ]
    
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=messages
    )
    
    return response.choices[0].message.content
```

### 代码实现：会话摘要自动生成

```python
from openai import OpenAI

client = OpenAI()

def summarize_conversation(conversation: list, max_tokens: 500) -> str:
    """自动摘要会话，提取关键信息"""
    prompt = """请分析以下对话，提取关键信息并生成摘要：

要求：
1. 提取用户的主要需求和目标
2. 记录已完成的操作和结果
3. 记录未完成的任务和下一步
4. 提取用户的重要偏好和信息

对话内容：
{conversation}

请用简洁的要点格式输出摘要："""
    
    formatted_prompt = prompt.format(
        conversation="\n".join([f"{msg['role']}: {msg['content']}" for msg in conversation])
    )
    
    response = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": formatted_prompt}],
        max_tokens=max_tokens
    )
    
    return response.choices[0].message.content
```

## 三、自主决策：Agent的思考

### 什么是自主决策？

Agent不仅响应指令，还能**分析情况、制定计划、选择最优行动**。

### ReAct模式：推理 + 行动

ReAct（Reasoning + Acting）是最经典的Agent决策框架：

```python
from openai import OpenAI
import json

client = OpenAI()

tools = [
    {
        "type": "function",
        "function": {
            "name": "search_code",
            "description": "搜索代码示例",
            "parameters": {
                "type": "object",
                "properties": {
                    "language": {"type": "string"},
                    "task": {"type": "string"}
                }
            }
        }
    },
    {
        "type": "function",
        "function": {
            "name": "write_file",
            "description": "写入文件",
            "parameters": {
                "type": "object",
                "properties": {
                    "path": {"type": "string"},
                    "content": {"type": "string"}
                }
            }
        }
    },
    {
        "type": "function",
        "function": {
            "name": "run_command",
            "description": "执行命令行",
            "parameters": {
                "type": "object",
                "properties": {
                    "command": {"type": "string"}
                }
            }
        }
    }
]

def react_agent(task: str, max_iterations: int = 10):
    """ReAct Agent主循环"""
    messages = [
        {"role": "system", "content": """你是一个智能编程助手。你需要：
1. 理解用户需求
2. 制定执行计划
3. 调用工具完成任务
4. 验证结果

每次只执行一个步骤，完成后汇报结果。"""},
        {"role": "user", "content": task}
    ]
    
    for i in range(max_iterations):
        # 1. 思考并决策
        response = client.chat.completions.create(
            model="gpt-4o",
            messages=messages,
            tools=tools
        )
        
        message = response.choices[0].message
        
        # 2. 如果需要调用工具
        if message.tool_calls:
            messages.append(message)
            
            for call in message.tool_calls:
                tool_name = call.function.name
                args = json.loads(call.function.arguments)
                
                # 3. 执行工具
                if tool_name == "search_code":
                    result = search_code(**args)
                elif tool_name == "write_file":
                    result = write_file(**args)
                elif tool_name == "run_command":
                    result = run_command(**args)
                
                # 4. 记录结果
                messages.append({
                    "role": "tool",
                    "tool_call_id": call.id,
                    "content": str(result)
                })
        else:
            # 5. 输出最终结果
            return message.content
    
    return "任务超时"
```

## 四、自主执行：Agent的手脚

### 什么是自主执行？

Agent能够**真正执行操作**，而不是只给建议。

### 工具生态集成

```python
import subprocess
import requests
from pathlib import Path

class ExecutionEngine:
    """执行引擎 - Agent的行动能力"""
    
    def __init__(self):
        self.tool_registry = {
            "bash": self.run_bash,
            "python": self.run_python,
            "http": self.http_request,
            "file_write": self.write_file,
            "file_read": self.read_file,
            "web_search": self.web_search,
        }
    
    def execute(self, tool_name: str, params: dict) -> dict:
        """执行工具"""
        if tool_name not in self.tool_registry:
            return {"success": False, "error": f"Unknown tool: {tool_name}"}
        
        try:
            result = self.tool_registry[tool_name](**params)
            return {"success": True, "result": result}
        except Exception as e:
            return {"success": False, "error": str(e)}
    
    def run_bash(self, command: str, timeout: int = 60) -> str:
        """执行bash命令"""
        result = subprocess.run(
            command,
            shell=True,
            capture_output=True,
            text=True,
            timeout=timeout
        )
        return result.stdout if result.returncode == 0 else result.stderr
    
    def http_request(self, method: str, url: str, **kwargs) -> dict:
        """发送HTTP请求"""
        response = requests.request(method, url, **kwargs)
        return {"status": response.status_code, "body": response.text}
    
    def write_file(self, path: str, content: str) -> str:
        """写入文件"""
        Path(path).parent.mkdir(parents=True, exist_ok=True)
        Path(path).write_text(content)
        return f"File written: {path}"
    
    def read_file(self, path: str) -> str:
        """读取文件"""
        return Path(path).read_text()
```

### 工作流编排

```python
from typing import Callable, Any

class Workflow:
    """工作流编排器"""
    
    def __init__(self):
        self.steps: list[dict] = []
    
    def add_step(self, name: str, action: Callable, condition: str = None):
        """添加工作流步骤"""
        self.steps.append({
            "name": name,
            "action": action,
            "condition": condition
        })
    
    def execute(self, context: dict) -> dict:
        """执行工作流"""
        results = []
        
        for step in self.steps:
            # 检查条件
            if step["condition"]:
                if not self.evaluate_condition(step["condition"], context):
                    continue
            
            # 执行步骤
            try:
                result = step["action"](context)
                context[step["name"]] = result
                results.append({"step": step["name"], "success": True, "result": result})
            except Exception as e:
                results.append({"step": step["name"], "success": False, "error": str(e)})
                break
        
        return {"context": context, "results": results}

# 使用示例
workflow = Workflow()

workflow.add_step("search", search_knowledge_base)
workflow.add_step("generate", generate_code)
```

## 五、自主交互：Agent的社交

### 什么是自主交互？

Agent能够**主动与用户、环境、其他Agent进行有效沟通**。

### 多轮对话管理

```python
class ConversationManager:
    """对话管理器"""
    
    def __init__(self, max_turns: int = 20):
        self.max_turns = max_turns
        self.conversation_history = []
        self.context = {}
    
    def add_turn(self, role: str, content: str):
        """添加对话轮次"""
        self.conversation_history.append({
            "role": role,
            "content": content
        })
        
        # 限制历史长度
        if len(self.conversation_history) > self.max_turns:
            # 压缩早期对话
            self.compress_history()
    
    def compress_history(self):
        """压缩历史对话"""
        if len(self.conversation_history) <= 6:
            return
        
        # 保留最近6轮 + 摘要早期
        recent = self.conversation_history[-6:]
        early_summary = summarize_conversation(self.conversation_history[:-6])
        
        self.conversation_history = [
            {"role": "system", "content": f"早期对话摘要：{early_summary}"}
        ] + recent
    
    def get_context_for_llm(self) -> list:
        """获取发送给LLM的上下文"""
        return self.conversation_history
```

### 完整代码示例

```python
from openai import OpenAI
from typing import Literal
import json

class CompleteAgent:
    """完整的AI Agent系统"""
    
    def __init__(self):
        self.client = OpenAI()
        self.memory = MemorySystem()
        self.tools = self._init_tools()
        self.conversation = ConversationManager()
    
    def _init_tools(self) -> list:
        return [
            {
                "type": "function",
                "function": {
                    "name": "web_search",
                    "description": "搜索互联网",
                    "parameters": {
                        "type": "object",
                        "properties": {
                            "query": {"type": "string"}
                        }
                    }
                }
            },
            {
                "type": "function",
                "function": {
                    "name": "code_interpreter",
                    "description": "执行Python代码",
                    "parameters": {
                        "type": "object",
                        "properties": {
                            "code": {"type": "string"}
                        }
                    }
                }
            },
            {
                "type": "function",
                "function": {
                    "name": "retrieve_knowledge",
                    "description": "检索知识库",
                    "parameters": {
                        "type": "object",
                        "properties": {
                            "query": {"type": "string"}
                        }
                    }
                }
            }
        ]
    
    def chat(self, user_input: str) -> str:
        """主对话入口"""
        # 1. 感知：获取用户输入
        self.conversation.add_turn("user", user_input)
        
        # 2. 记忆：检索相关上下文
        relevant_memory = self.memory.retrieve(user_input)
        
        # 3. 决策：调用LLM进行推理
        messages = [
            {"role": "system", "content": f"相关记忆：{relevant_memory}"}
        ] + self.conversation.get_history()
        
        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=messages,
            tools=self.tools
        )
        
        message = response.choices[0].message
        self.conversation.add_turn("assistant", message.content)
        
        # 4. 执行：如果需要调用工具
        if message.tool_calls:
            for call in message.tool_calls:
                result = self._execute_tool(call)
                self.conversation.add_turn("tool", f"{call.function.name}: {result}")
        
        # 5. 记忆：更新记忆系统
        self.memory.store(user_input, message.content)
        
        return message.content or "任务完成"
    
    def _execute_tool(self, tool_call) -> str:
        """执行工具"""
        name = tool_call.function.name
        args = json.loads(tool_call.function.arguments)
        
        if name == "web_search":
            return self._web_search(args["query"])
        elif name == "code_interpreter":
            return self._run_code(args["code"])
        elif name == "retrieve_knowledge":
            return self.memory.retrieve(args["query"])
        
        return "Unknown tool"

# 使用示例

# 单轮对话
response = agent.chat("帮我查一下最新的AI Agent框架有哪些")

# 多轮对话

agent.chat("部署到服务器上")
```

## 总结

构建真正的AI Agent需要五大核心能力：

| 能力 | 关键词 | 技术栈 |
|------|--------|--------|
| **自主感知** | 工具调用、多模态 | Function Calling、Vision API |
| **自主记忆** | RAG、向量数据库 | ChromaDB、Milvus、Pinecone |
| **自主决策** | ReAct、CoT | Prompt Engineering |
| **自主执行** | 代码执行、API调用 | Python Runtime、Shell |
| **自主交互** | 多轮对话、主动推送 | Conversation Manager |

掌握了这些技术，你就能构建出真正智能、能够自主工作的AI Agent系统。
