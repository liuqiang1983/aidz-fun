# Coze 入门教程：从 0 到 1 搭建你的第一个 AI Agent

Coze 是字节跳动推出的 AI Agent 开发平台，支持快速创建、部署和管理 AI Agent。本教程将带你从零开始，在 30 分钟内完成第一个 Agent 的创建和部署。

## 什么是 Coze？

Coze 是一个无需编程基础即可创建 AI 应用 的平台，提供了：

- 可视化的 Bot 编辑器
- 丰富的插件市场
- 多平台发布能力
- 知识库管理

## 前置准备

在开始之前，你需要准备：

1. **Coze 账号**：访问 [coze.cn](https://coze.cn) 注册
2. **Coze API Token**：在设置页面获取

<PromptBlock content={`# 获取 Coze API Token
1. 登录 Coze 控制台
2. 点击右上角头像 → 设置
3. 在 API 密钥页面创建新密钥
4. 复制并妥善保存你的 Token`} title="准备工作" />

## 创建第一个 Bot

### 第一步：创建 Bot

1. 登录 Coze 控制台
2. 点击「创建 Bot」按钮
3. 填写 Bot 名称和描述

<CodeBlock language="json" code={`{
  "name": "我的助手",
  "description": "一个可以回答问题的 AI 助手",
  "icon": "🤖"
}`} />

### 第二步：配置 Bot 角色

在「人设与回复逻辑」中定义 Bot 的角色：

<PromptBlock content={`你是一个专业、高效的 AI 助手。

你的特点：
- 回答准确、简洁
- 善于用列表和表格组织信息
- 遇到不确定的问题会坦诚告知

你的专长领域：
- AI 工具使用
- 工作效率提升
- 常见问题解答`} title="Bot 角色设定" />

### 第三步：添加插件

Coze 提供了丰富的插件，可以扩展 Bot 的能力：

1. 点击「插件」标签
2. 在插件市场中选择需要的插件
3. 推荐新手添加：搜索插件、天气查询

### 第四步：发布 Bot

配置完成后，点击「发布」按钮即可上线你的 Bot。

## 进阶技巧

### 使用知识库

如果想让 Bot 回答特定领域的问题，可以添加知识库：

1. 在「知识库」页面创建知识库
2. 上传文档（支持 PDF、TXT、Word）
3. 在 Bot 配置中关联知识库

### 配置工作流

对于复杂任务，可以使用工作流：

<CodeBlock language="yaml" code={`工作流名称: 文章处理流程
步骤:
  1. 用户输入文章链接
  2. 读取文章内容
  3. 提取关键信息
  4. 生成摘要
  5. 输出结果`} />

## 常见问题

### Q: Bot 回复速度慢怎么办？

A: 可以尝试以下优化：
- 减少上下文长度
- 关闭不必要的插件
- 选择更快的模型

### Q: 如何让 Bot 回答更准确？

A: 建议：
- 完善人设描述
- 添加示例对话
- 使用知识库补充专业信息
