# Coze入门教程：从零开始创建你的第一个AI Bot

Coze是字节跳动推出的AI Bot开发平台，无需编程基础即可创建智能对话机器人。本教程将带你从零开始创建你的第一个AI Bot。

## 前置条件

- 一个Coze账号（可使用抖音或手机号注册）
- 基本的互联网使用能力
- 约15分钟时间

## 第一步：注册并登录Coze

1. 访问 [Coze官网](https://www.coze.com)
2. 点击右上角"登录"按钮
3. 选择抖音登录或手机号登录
4. 完成账号注册流程

## 第二步：创建新Bot

1. 登录后，点击左侧导航栏的"Bot"选项
2. 点击右上角"创建Bot"按钮
3. 填写Bot基本信息：
   - **名称**：给你的Bot起个名字
   - **简介**：简单描述Bot的功能
   - **图标**：上传或选择一个图标

## 第三步：配置Bot提示词

在"人设与回复逻辑"区域，编写你的Bot提示词：

```text
你是一个友好的AI助手，专门帮助用户解答关于[你的领域]的问题。

你的特点：
- 回答简洁明了
- 态度友好热情
- 优先提供实用建议
- 不确定时会诚实说明
```

## 第四步：测试Bot

1. 点击右侧"预览"按钮
2. 在对话框中输入测试问题
3. 查看Bot的回复是否符合预期
4. 根据测试结果调整提示词

## 第五步：发布Bot

1. 点击右上角"发布"按钮
2. 选择发布渠道（飞书、微信等）
3. 完成发布配置
4. 等待审核通过

## 常见问题

### Q: Bot回复不准确怎么办？

检查并优化你的提示词，使其更加具体和明确。

### Q: 如何让Bot记住上下文？

Coze默认支持多轮对话，你可以在提示词中明确要求Bot记住之前的对话内容。

### Q: 发布审核需要多久？

通常1-3个工作日，具体时间取决于发布渠道。
