# Dify本地部署完整指南

## 环境要求

- Docker 20.0+
- Docker Compose 2.0+
- 内存：至少4GB
- 磁盘：至少20GB

## 部署步骤

1. 下载Dify源码
2. 配置环境变量
3. 启动Docker服务
4. 访问Web界面

## 配置说明

在.env文件中配置密钥和URL。

## 模型配置

支持OpenAI、Claude、文心一言等多种模型。
