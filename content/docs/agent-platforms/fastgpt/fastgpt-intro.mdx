# FastGPT快速入门

## 什么是FastGPT

FastGPT是一个基于LLM构建的知识库问答平台，支持导入文档、自动分段、向量化检索。

## 核心功能

- 知识库管理
- 自动向量化
- 多模型支持
- API接口
