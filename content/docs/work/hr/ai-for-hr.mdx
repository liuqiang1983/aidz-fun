## 概述

HR 是企业最重要的"人才入口"和"员工守护者"，但繁琐的事务性工作让很多 HR 难以聚焦于真正重要的事——人才战略和文化建设。

AI 正在为 HR 领域带来深刻变革。从智能筛选简历到面试辅助，从员工关系到数据分析，AI 工具正在帮助 HR 从业者从繁琐中解放出来。

作为 HR，你是否被这些工作困扰？

- 📄 简历太多，看不过来
- 📅 面试安排耗时耗力
- 📊 绩效评估主观性太强
- 👥 员工问题咨询量大
- 📚 培训内容制作耗时

**本文将分享 HR AI 工作流的实战经验**，涵盖招聘、人才发展、员工关系、HR 分析四大场景。

## 智能招聘 AI 实战 🔍

### JD 优化与吸引力提升

JD 写不好，候选人都不来。AI 帮你写出让候选人眼前一亮的职位描述：

```markdown
# JD 优化 Prompt
请帮我优化以下职位描述：

原始 JD：
[粘贴原始 JD]

目标候选人画像：
- 资历：[初级/中级/高级]
- 偏好：[创业公司/大厂/远程]
- 关注点：[成长/薪资/文化]

请优化：
1. 职位标题（更吸引人）
2. 开头简介（30字内，突出亮点）
3. 职责描述（5-7条，动词开头）
4. 任职要求（5条以内，分层）
5. 福利待遇（5-8条）
6. 团队文化描述

风格要求：专业但有温度，避免"催命式"JD
```

### 简历筛选效率提升

面对海量简历，AI 可以帮你快速识别优质候选人：

```markdown
# 简历筛选 Prompt
请帮我筛选简历：

招聘岗位：[职位名称]
核心要求：[3-5个硬性要求]
加分项：[优先考虑的因素]

简历内容：
[粘贴简历全文]

请给出：
1. 与岗位的匹配度评分（1-10）
2. 优势分析
3. 风险点（如有）
4. 建议面试问题（3个）
5. 是否进入面试（推荐/待定/不建议）
```

### 候选人评估报告生成

```markdown
# 候选人评估 Prompt
面试信息：
- 候选人：[姓名]
- 应聘岗位：[岗位名]
- 面试形式：[视频/现场/笔试]
- 面试时长：[X分钟]

面试要点记录：
[粘贴面试记录]

请生成面试评估报告：
1. 综合评价（3-5句话）
2. 能力维度评分
   - 专业能力
   - 沟通表达
   - 思维逻辑
   - 文化匹配
   - 发展潜力
3. 关键亮点
4. 主要顾虑（如有）
5. 录用建议
```

### 背调问题清单生成

```markdown
# 背调问题设计
候选人：[姓名]
应聘岗位：[岗位]
前公司：[公司名]
担任职位：[职位]
汇报对象：[职位]
工作时长：[起止]

请设计背调访谈问题：
1. 工作表现（2-3个问题）
2. 职业素养（2-3个问题）
3. 团队协作（2-3个问题）
4. 管理能力（如适用）（2-3个问题）
5. 其他关注点（开放性问题）

每个问题后标注：验证重点
```

## 员工管理 AI 实战 👥

### 入职流程自动化

AI 可以帮助设计智能入职流程：

```markdown
# 入职流程设计
新员工信息：
- 姓名：[姓名]
- 岗位：[岗位]
- 部门：[部门]
- 入职日期：[日期]
- 工作地点：[地点]

请设计入职流程：
1. 入职前准备清单（HR侧）
2. 入职第一天安排（上午/下午）
3. 第一周学习计划
4. 第一个月目标设定
5. 导师/Buddy 任务清单
6. 需要提前准备的账号和权限
```

### 员工手册生成

```markdown
# 员工手册 Prompt
公司信息：
- 公司名称：[名称]
- 行业：[行业]
- 公司规模：[规模]
- 企业文化：[关键词]

请生成员工手册大纲，包含：
1. 公司简介与价值观
2. 入职须知（入职准备、第一天）
3. 工作时间与考勤
4. 薪酬福利
5. 假期制度
6. 行为规范
7. 绩效考核
8. 离职流程

语言风格：专业、温暖、易读
字数要求：每章节 100-200 字
```

### 绩效评估辅助

绩效评估头疼？AI 帮你：

```markdown
# 绩效评估 Prompt
员工信息：
- 姓名：[姓名]
- 岗位：[岗位]
- 评估周期：[年/半年/季度]
- 直接上级：[姓名]

关键产出：
[列出评估周期内主要成果]

目标完成情况：
[列出年初/期初设定的目标及完成情况]

请生成：
1. 绩效评估总结（基于产出的客观描述）
2. 能力评估维度
3. 优点认可点
4. 改进建议（2-3条）
5. 下一步发展计划
6. 综合评级建议

注意：基于事实，客观公正，避免主观情绪
```

## 培训发展 AI 实战 📚

### 培训需求分析

```markdown
# 培训需求分析
组织背景：
- 部门：[部门]
- 当前业务挑战：[描述]
- 团队规模：[X人]

成员情况：
- 平均司龄：[X年]
- 能力水平分布：[初级/中级/高级比例]
- 最近培训记录：[如有]

请分析：
1. 当前最需要提升的能力项（Top3）
2. 培训优先级排序
3. 推荐的培训形式（线上/线下/实战）
4. 预期效果衡量指标
```

### 培训内容生成

```markdown
# 培训内容生成
培训主题：[主题]
目标学员：[人群]
培训时长：[X分钟]
培训目标：[学完能做什么]

请设计培训内容：
1. 课程大纲
2. 每个模块的时长分配
3. 关键知识点（每模块 3-5 个）
4. 互动环节设计
5. 案例/练习素材
6. 效果检验方式
7. 课后作业/行动计划

格式：可用 Markdown 输出，便于直接制作 PPT
```

### 新经理 onboarding

```markdown
# 新经理培训计划
新晋管理者信息：
- 姓名：[姓名]
- 原岗位：[岗位]
- 新岗位：[管理岗位]
- 团队规模：[X人]
- 团队成员构成：[简要描述]

请设计 30-60-90 天计划：
1. 第一个月重点：融入 + 观察
2. 第二个月重点：建立 + 学习
3. 第三个月重点：承担 + 优化

每个阶段包含：
- 关键任务
- 学习主题
- 需要建立的会议/1:1
- 成功指标
```

## 员工沟通 AI 辅助 💬

### 沟通政策文档

```markdown
# 政策文档生成
政策类型：[如远程办公/病假/年假/晋升等]
公司背景：[行业/规模/文化]

请生成政策文档：
1. 政策目的（1-2句话）
2. 适用范围
3. 详细条款（清晰、无歧义）
4. 申请流程
5. 审批权限
6. 常见问题 FAQ
7. 联系方式

风格：正式但易读，避免法律术语堆砌
```

### 团队沟通优化

```markdown
# 沟通场景优化
场景：[全员通知/绩效反馈/变动通知/表扬信]
背景：[简要说明]

请撰写：
1. 沟通草稿
2. 不同渠道的适配版本
   - 邮件版
   - 即时消息版
   - 会议口述要点
3. 预期可能的反馈及应对
```

### 离职沟通处理

```markdown
请给出：
1. 离职面谈准备清单
2. 面谈话术建议
3. 挽留策略（如适用）
4. 风险预警

```
