## 写在前面

自媒体是最适合用 AI 提效的行业之一。

**为什么？**

- 内容生产是核心工作（文字、图片、视频）
- 需要持续输出（日更、周更压力）
- 多平台运营（公众号、小红书、抖音、B站）
- 数据驱动决策（选题、优化、变现）

AI 可以帮你：

- ✅ 选题策划：热点追踪、爆款分析、选题库生成
- ✅ 内容创作：文案写作、脚本生成、标题优化
- ✅ 视频制作：脚本分镜、配音字幕、封面设计
- ✅ 数据分析：爆款拆解、粉丝画像、变现优化
- ✅ 多平台适配：一键改写、格式转换

**实测效果**：一个全职自媒体人，用 AI 后内容产出效率提升 60%，多平台运营时间减少 70%。

## 一、选题策划：从拍脑袋到数据驱动

### 场景 1：热点追踪

**Prompt 模板**：
```
请帮我分析今天的热点话题，筛选出适合 [你的领域] 的选题：

热点列表：
[粘贴微博热搜/抖音热榜/知乎热榜]

我的领域：[XX]
目标用户：[XX]
内容风格：[XX]

请输出：
1. 可蹭的热点（3-5 个）
2. 每个热点的切入角度
3. 预估爆款概率
4. 建议发布时间
```

### 场景 2：爆款拆解

**Prompt 模板**：
```
请拆解这篇爆款内容：

标题：[XX]
内容：[粘贴]

请分析：
1. 标题的爆款元素（数字、冲突、悬念等）
2. 开头的钩子设计
3. 内容结构（起承转合）
4. 情绪曲线
5. 可复用的套路
```

### 场景 3：选题库生成

**Prompt 模板**：
```
请帮我生成 [领域] 的选题库：

我的领域：[XX]
目标用户：[XX]
内容形式：[图文/视频/...]

请输出 50 个选题，分为：
1. 干货类（15 个）
2. 观点类（10 个）
3. 故事类（10 个）
4. 清单类（10 个）
5. 互动类（5 个）

每个选题包含：
- 标题
- 核心观点
- 预估爆款指数
```

## 二、内容创作：从憋不出到批量产

### 场景 1：公众号文章

**Prompt 模板**：
```
请帮我写一篇公众号文章：

主题：[XX]
目标读者：[XX]
字数：[XX]
风格：[XX]

1. 标题（3 个备选）
2. 开头钩子（50 字以内）
3. 正文（3-5 个小标题）
4. 结尾行动召唤

要求：
- 口语化、有网感
- 适当使用表情符号
- 段落简短（3-5 行）
- 金句密度高
```

```
1. 标题（5 个备选，25 字以内）
```

## 九、常见问题

**Q：AI 生成的内容会被平台判定为抄袭吗？**
A：不会。AI 生成的是原创内容，但建议你进行二次加工，加入个人观点和案例。

**Q：用 AI 写的内容质量如何？**
A：AI 生成的是初稿，需要你优化。核心观点、案例、情感表达需要人工打磨。

**Q：AI 能替代自媒体人吗？**
A：不能。AI 提升效率，但选题判断、内容调性、粉丝互动需要人来做。

**Q：哪些内容适合用 AI？**
A：干货类、清单类、资讯类适合。故事类、观点类、情感类需要更多人工。

**Q：用 AI 会影响账号权重吗？**
A：不会。只要内容质量好、用户喜欢，平台不会因为用了 AI 而降权。

## 最后一句

AI 不是来替代自媒体人的，是来**替代自媒体人的重复劳动**的。

会用 AI 的自媒体人，把时间花在：
- 选题判断而不是憋内容
- 粉丝互动而不是机械运营
- 商业变现而不是重复劳动

**记住**：AI 是工具，你是创作者。用对了，效率提升 60%；用错了，内容同质化严重。
