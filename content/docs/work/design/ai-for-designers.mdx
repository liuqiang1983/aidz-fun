作为设计师，你是否面临这些挑战？

## Midjourney 创意工作流 🎨

### 基础设置与参数指南

```markdown
基础命令格式：
/imagine prompt: [描述词] --[参数]

常用参数：
- --ar 16:9      画面比例
- --v 6          版本选择（1-6）
- --s 750        风格化程度（0-1000）
- --q 1          质量（0.25/1/2）
- --niji 6       动漫风格专用
- --style raw    写实风格
```

### 场景一：App 概念设计

**Prompt 模板**：

```markdown
[产品类型] app interface, [核心功能], [设计风格],
[配色偏好], [参考应用], UI/UX design, mobile screen,
glass morphism, [光效描述], 4K, --ar 9:16 --v 6
```

**实战案例**：

```markdown
fitness tracking app interface, heart rate monitoring,
workout history, minimalist design, soft pastel colors,
calm atmosphere, modern UI, mobile screen, glass morphism,
soft gradient glow, 4K --ar 9:16 --v 6 --s 800
```

### 场景二：Logo 与品牌视觉

```markdown
# 极简风格 Logo
minimalist [品牌类型] logo, [核心元素], [配色],
vector style, white background, clean lines, brand identity,
logo design --ar 1:1 --v 6 --niji 5

# 实际案例
minimalist coffee shop logo, coffee bean silhouette,
earth tones, vector style, white background,
clean lines, brand identity, logo design
--ar 1:1 --v 6 --niji 5
```

### 场景三：产品海报与配图

```markdown
[产品名] product photography, [使用场景],
[光线风格], [配色基调], [情感氛围],
editorial photography, high detail, 8K --ar 3:4 --v 6
```

**进阶技巧**：

| 技巧 | 作用 | 示例 |
|-----|------|------|
| --cref | 参考角色特征 | --cref [图片URL] |
| --sref | 参考风格 | --sref [图片URL] |
| 负面提示 | 排除不需要元素 | --no text, watermark |
| 权重调整 | 控制元素比重 | "dog:2 car:0.5" |

## Figma AI 智能设计 🔮

### Figma 内置 AI 功能

2024 年 Figma 全面拥抱 AI，带来多项重磅功能：

| 功能 | 用途 | 使用方式 |
|-----|------|---------|
| **Magical Replace** | 一键替换图片风格 | 选中图片 → AI 菜单 |
| **Smart Animate** | 智能动画过渡 | 选择元素 → AI 生成 |
| **Auto Layout** | 自适应布局 | 右键 → Add Auto Layout |
| **Component AI** | AI 生成组件 | 使用 / 快捷命令 |
| **Analyze** | 设计稿分析 | 选中画板 → Analyze |

### 快速生成 UI 组件

**Step 1**：在 Figma 中新建画板

**Step 2**：输入快捷键 `Ctrl+/` 打开命令面板

**Step 3**：输入 `Make component` 或 `AI`

**Step 4**：用自然语言描述想要的组件

```markdown
示例 Prompt：
"创建一个用户头像组件，支持不同尺寸(s/m/l)，
圆形和方形，默认显示蓝色渐变背景和首字母"
```

### AI 辅助设计系统

```markdown
# 设计 Token 生成
"为我的 App 生成一套设计 token：
- 主色：科技蓝
- 辅助色：暖橙
- 字体：思源黑体
- 圆角：12px
- 阴影：柔和风格"

# 图标集生成
"生成一套 24x24 的天气图标集，包含：
晴天、多云、雨天、雪天、雷电"
```

## AI 辅助设计实战案例 🎯

### 案例一：App 启动页设计

**需求**：为一款冥想 App 设计 5 个启动页插图

**传统流程**：1-2 天
**AI 流程**：2-3 小时

**Midjourney Prompt 示例**：

```markdown
# 第1张：宁静场景
serene meditation illustration, person sitting in lotus position,
mountain landscape, sunrise, soft orange and purple gradient sky,
peaceful atmosphere, vector style, minimalist, flat illustration
--ar 9:16 --v 6 --style raw

# 第2张：呼吸可视化
abstract meditation app illustration, gentle breathing visualization,
concentric circles expanding, soft teal and white colors,
flowing water effect, calming mood, modern UI style
--ar 9:16 --v 6 --s 600
```

### 案例二：图标集批量生成

**工具推荐**：IconifyAI、Magic Icon

**Prompt 模板**：

```markdown
Create a set of [数量] [主题] icons, 24x24, line style,
[线条粗细] stroke width, [配色], uniform style,
SVG format, clean design, consistent proportions
```

### Prompt 优化策略

| 技巧 | 说明 | 效果 |
|-----|------|------|
| 风格词 | 加入艺术风格描述 | 更精准的风格控制 |
| 质量词 | 添加 4K、HD 等 | 提升清晰度 |
| 光线词 | 描述主光源 | 氛围感 |
| 构图词 | 指定视角 | 画面张力 |

### 设计规范 Prompt 模板 📝

```markdown
# Role: 资深 UI/UX 设计师

# Task: 为 [产品类型] 设计 [组件类型]

# Context:
- 产品定位：[目标用户、核心价值]
- 品牌调性：[风格关键词]
- 设计规范：[已有规范摘要]

# Requirements:
1. 组件需要包含 [状态列表]
2. 支持 [尺寸/平台]
3. 符合 [设计系统版本] 规范

# Output:
- 设计建议
- 实现要点
- 可参考案例
```
