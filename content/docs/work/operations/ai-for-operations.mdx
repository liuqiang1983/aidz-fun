## 概述

运营是互联网公司的"万能钥匙"——写文案、做活动、分析数据、维护用户，样样都要会。AI 工具的出现，让运营人终于可以从繁琐重复的工作中解放出来。

作为运营，你是否经常被这些工作淹没？

- 📝 每天写不完的文案和选题
- 📊 数据报表分析耗时又枯燥
- 👥 用户消息回复不过来
- 📅 活动策划灵感枯竭

**好消息**：这些工作 AI 都能帮你做得更快更好。本文将分享运营 AI 工作流的实战经验，涵盖内容营销、用户运营、活动策划、数据分析四大场景。

## 内容营销 AI 工作流 📝

### 选题策划与爆款分析

**痛点**：不知道写什么、怎么判断选题热度

**AI 辅助流程**：

```markdown
# 选题分析 Prompt
请分析以下选题的热度和潜力：

1. 目标平台：[小红书/公众号/抖音]
2. 目标用户：[人群描述]
3. 选题列表：
   - [选题A]
   - [选题B]
   - [选题C]

请从以下维度评分（1-10分）：
- 用户关注度
- 搜索热度
- 竞争程度
- 商业价值
- 可执行性

并给出每个选题的优化建议。
```

### 爆款文案结构拆解

使用 AI 分析竞品爆款文章的结构：

```markdown
# 爆款文章分析
请分析以下这篇爆款文章的结构和成功要素：

[粘贴文章内容或链接]

分析维度：
1. 标题结构（悬念/数字/情感/利益）
2. 开头写法（3秒法则）
3. 内容框架（故事型/清单型/对比型）
4. 结尾引导（互动/转发/收藏）
5. 金句摘录

基于以上分析，请给出一个可复用的内容模板。
```

### 多平台内容适配

一份内容，多平台分发是运营的基本功。AI 可以帮你快速适配：

```markdown
# 内容多平台适配
原始内容：[粘贴文章/视频脚本]

请帮我改写成以下格式：

1. 小红书图文（600-800字）：
   - 开头：痛点场景引入
   - 中间：3-5个实用要点
   - 结尾：互动引导（评论区）

2. 微信公众号（1000-1500字）：
   - 深度扩展，增加案例
   - 保持专业调性

3. 微博/短文案（100字以内）：
   - 提炼核心观点
   - 添加热门话题标签

4. 知乎回答（结构化）：
   - 先说结论
   - 分点论述
   - 案例支撑
```

## 数据分析 AI 实战 📊

### 报表解读与洞察

**痛点**：数据量大，不知道从哪看起

**AI 辅助流程**：

```markdown
# 数据分析 Prompt
我的[产品名]最近数据表现如下：

| 指标 | 上周 | 本周 | 变化 |
|-----|-----|-----|-----|
| DAU | 10000 | 8500 | -15% |
| 留存率 | 35% | 32% | -3% |
| 转化率 | 5% | 4.2% | -0.8% |

请分析：
1. 主要下跌原因可能是什么？
2. 需要重点关注哪些指标？
3. 建议下一步行动方案？
```

### SQL 查询生成

不会写 SQL？让 AI 帮你：

```markdown
# SQL 生成
请帮我生成查询[需求描述]的 SQL：

数据库表结构：
- users: id, name, created_at, channel
- events: id, user_id, event_name, created_at
- orders: id, user_id, amount, created_at

需求：[详细描述]
例如："统计过去7天每日新增用户数，按渠道分组"
```

### 数据可视化建议

```markdown
# 可视化建议
我有以下数据：

[描述数据内容]

请推荐：
1. 最适合的可视化图表类型
2. 关键指标的展示方式
3. 可以突出展示的洞察点
```

### 用户分层 Prompt

```markdown
# 用户分群分析
根据以下用户行为数据，请帮我进行用户分层：

[粘贴用户行为数据]

请分析：
1. 用户可以分为哪几类？
2. 每类用户的特征是什么？
3. 针对不同层级用户，运营策略建议？
4. 重点投入的优质用户画像描述？
```

### 推送文案 A/B 测试

```markdown
# 推送文案优化
产品：[App名称]
推送场景：[活跃召回/促活/促销]

原始文案：[原文案]

请生成 5 个替代文案，要求：
1. 包含[emoji/数字/悬念]元素
2. 每条突出不同卖点
3. 长度控制在 20 字以内
4. 标注每条文案的设计意图
```

## 活动策划 AI 辅助 🎯

### 活动方案生成

```markdown
# 活动策划 Prompt
请为[产品名]策划一场[活动类型]：

基本信息：
- 活动目标：[拉新/促活/转化/品牌]
- 预算范围：[X万元]
- 活动时间：[起止日期]
- 目标用户：[人群描述]

请提供：
1. 活动主题和 slogan
2. 核心玩法设计（3个方案对比）
3. 用户参与路径
4. 关键时间节点
5. 预期效果指标
6. 风险点和备选方案
```

### 活动文案撰写

```markdown
# 活动文案生成
活动：[活动名称]
背景：[简要说明]

请撰写：
1. 活动 Banner 文案（主标题+副标题）
2. 站内弹窗文案
3. Push 推送文案（3条备选）
4. 用户分享话术
5. 结束后的感谢文案

风格要求：[正式/活泼/简洁/情感化]
```
