## 概述

创业是一场九死一生的冒险。作为创始人，你需要是全能选手——商业策划、产品开发、市场营销、财务管理，样样都要懂。

AI 的出现，让"一个人就是一支队伍"成为可能。从商业洞察到产品 MVP，从内容营销到客户服务，AI 工具正在大幅降低创业的门槛和成本。

作为创业者或副业者，你是否面临这些挑战？

- 💡 有想法但不知道怎么验证
- 💰 预算有限，请不起团队
- 📱 产品、运营、营销全要自己做
- ⏰ 时间不够用，恨不得三头六臂
- 📊 不懂技术，不知道怎么落地

**本文将分享创业者 AI 工作流的实战经验**，涵盖商业策划、产品开发、市场营销、运营管理四大场景，助你用最小成本快速验证想法。

## 商业策划 AI 实战 💡

### 商业想法验证

创业第一步：验证想法是否靠谱。AI 帮你做快速的市场调研：

```markdown
# 商业想法验证 Prompt
我想验证一个创业想法：

想法概述：[简要描述你的产品/服务]

目标用户：[谁会用？]
解决的问题：[痛点是什么？]
解决方案：[你怎么解决？]
商业模式：[怎么赚钱？]

请帮我分析：
1. 市场规模（TAM/SAM/SOM）
2. 目标用户画像（3-5个细分人群）
3. 主要竞品（3-5个）+ 差异化机会
4. 可能的风险和挑战
5. 验证这个想法需要回答的核心问题（5个）
6. 建议的验证方式（快速低成本）
```

### 竞品分析框架

知己知彼，百战不殆：

```markdown
# 竞品分析 Prompt
我的产品：[产品名称]
定位：[市场定位]

请帮我分析以下竞品：

竞品列表：
1. [竞品1名称] - [简要描述]
2. [竞品2名称] - [简要描述]
3. [竞品3名称] - [简要描述]

分析维度：
1. 定价策略
2. 核心卖点
3. 目标用户
4. 优势
5. 劣势
6. 用户评价（App Store/知乎/小红书）
7. 可能的差异化方向

输出格式：对比表格 + 每个竞品的详细分析
```

```markdown
# BP 辅助生成 Prompt
项目信息：

- 产品类型：[SaaS/电商/内容/平台等]

- 目标市场：[市场描述]
- 商业模式：[盈利方式]
- 团队情况：[团队背景]

请帮我生成商业计划书大纲，包含：
1. 执行摘要
2. 问题与痛点
3. 解决方案
4. 市场规模
5. 商业模式
6. 竞争分析
7. 营销策略
8. 团队介绍
9. 财务预测（3年）
10. 融资需求

每部分请提供：
- 核心要点（bullet points）
- 需要回答的关键问题
- 数据/案例支撑建议
```

## 产品开发 AI 实战 🚀

### MVP 规划与优先级

创业公司最大的敌人是完美主义。AI 帮你快速规划 MVP：

```markdown
# MVP 规划 Prompt
产品构想：[描述产品]
核心资源：[人力/资金/时间限制]

请帮我规划 MVP：
1. MVP 必须包含的核心功能（最多 5 个）
2. 功能优先级排序（MoSCoW 法则）
3. 每个功能的用户故事（User Story）
4. "不可做"清单（排除的功能）
5. 技术实现建议
6. 预计开发周期
7. 如何验证产品假设

重点：只做能验证核心价值假设的功能
```

### 用户故事与需求文档

```markdown
# 用户故事生成 Prompt
产品：[产品名称]
目标用户：[用户画像]

请生成用户故事（User Story）：
格式：As a [角色], I want to [功能], so that [价值]

需要覆盖：
1. 核心用户角色（3-5个）
2. 每个角色的主要用户故事（5-10个）
3. 边缘场景用户故事（2-3个）
4. 故事验收标准（Acceptance Criteria）

优先级：高（核心体验）/ 中（重要功能）/ 低（增强体验）
```

### 产品路线图规划

```markdown
# 产品路线图 Prompt
当前状态：[产品当前阶段]
团队规模：[X人]
可用资源：[时间/预算]

请规划 12 个月的产品路线图：
1. 每个季度的核心目标（OKR 格式）
2. 每个月的关键里程碑
3. 技术债务处理计划
4. 团队能力建设
5. 用户增长目标
6. 收入目标（如有）

输出格式：时间线表格 + 每个阶段的关键决策点
```

## 市场营销 AI 实战 📢

### 品牌定位与 Slogan

```markdown
# 品牌定位 Prompt
产品：[产品名称]
解决的问题：[痛点]
解决方案：[核心价值]
目标用户：[用户群]
差异化：[与竞品不同点]

请帮我设计：
1. 品牌定位陈述（1-2句话）
2. 品牌个性（3-5个关键词）
3. 品牌声音（语气/风格指南）
4. 核心 Slogan（3-5个备选）
5. tagline 背后的逻辑

风格：年轻化/专业/温暖/科技感
```

### 多渠道内容策略

一个人做营销？AI 帮你规划内容矩阵：

```markdown
# 内容策略 Prompt
产品：[产品名称]
目标用户：[用户画像]
主要渠道：[小红书/抖音/公众号/知乎等]

请设计内容策略：
1. 各平台内容定位（差异化）
2. 内容类型配比（如：干货 60%、软广 30%、人设 10%）
3. 每周发布频率建议
4. 内容选题库（每个平台 10 个）
5. 爆款内容模板
6. 评论区互动策略

重点：如何用最小精力产出最大效果
```

### 销售页/落地页文案

```markdown
# 销售页文案 Prompt
产品：[产品/服务名称]
目标：[获取销售/收集线索/试用]
核心卖点：[3-5个]
目标用户：[用户痛点和渴望]
定价：[价格/套餐]

请生成销售页文案：
1. 吸睛标题（3个备选）
2. 痛点共鸣段落
3. 解决方案呈现
4. 信任背书（案例/数据/评价）
5. 产品展示
6. 价格锚定与套餐设计
7. 紧迫感与 CTA
8. FAQ（5个常见问题）

字数：每个模块 50-200 字
风格：[销售型/信任型/紧迫型]
```

## 运营管理 AI 实战 📊

### 运营指标体系

数据驱动是创业公司的核心竞争力：

```markdown
# 指标体系设计
产品类型：[SaaS/电商/内容/平台等]
当前阶段：[0-1/规模化/成熟]
核心目标：[用户增长/收入/留存]

请设计运营指标体系：
1. 北极星指标（1个）
2. 链路指标（5-8个）
   - 获取指标
   - 激活指标
   - 留存指标
   - 收入指标
   - 推荐指标
3. 每个指标的定义
4. 目标值设定参考
5. 指标看板建议

格式：表格 + 公式 + 可视化建议
```

### 客服话术与 FAQ

```markdown
# 客服 FAQ 生成
产品：[产品名称]
常见问题类型：[使用问题/支付问题/功能咨询/投诉]

请生成 FAQ：
1. Top 20 常见问题
2. 每个问题的标准答案
3. 需要转人工的判断标准
4. 情绪安抚话术（如有）
5. 升级处理流程

风格：专业、耐心、友好
```

```markdown
# 周报模板 Prompt
请为 [阶段] 创业公司设计周报模板：

包含模块：
1. 本周核心成果（数据化）
2. 关键指标变化
3. 重点项目进展
4. 遇到的问题与解决方案
5. 下周重点计划

风格：简洁、数据驱动、重点突出

```

## 创业者常用 Prompt 模板库 📝

### 融资相关

```markdown
# 投资人问题准备
产品：[产品名称]
融资轮次：[种子轮/A轮等]
融资金额：[X万美元]
估值：[X万美元]

请帮我准备投资人常见问题及回答：
1. 产品与市场（5个问题）
2. 商业模式（5个问题）
3. 竞争壁垒（5个问题）
4. 团队（3个问题）
5. 融资用途（3个问题）

每个问题：问题 + 参考回答 + 注意事项
```

### 定价策略

```markdown
# 定价策略分析
产品：[产品名称]
成本结构：[成本描述]
竞品定价：[竞品价格]
目标用户：[用户支付意愿]

请分析：
1. 定价策略选择（渗透/撇脂/价值定价）
2. 价格区间建议
3. 定价模型（订阅/一次性/混合）
4. 套餐设计建议
5. 早期用户优惠策略
6. 涨价策略预案
```

### 危机处理

```markdown
# 危机公关 Prompt
危机情况：[描述]
影响范围：[用户/媒体/合作伙伴]
紧迫程度：[紧急/重要/一般]

请给出：
1. 立即行动清单（24小时内）
2. 对内沟通口径
3. 对外回应声明
4. 后续跟进措施
5. 预防类似问题的机制
```

**核心收获**：

- 💡 AI 帮你快速验证商业想法
- 🚀 MVP 开发效率翻倍
- 📢 内容营销不再是一个人战斗

1. 今天用 AI 验证你的一个想法
