## 写在前面

教师是最需要 AI 辅助的职业之一。

**为什么？**

- 备课耗时（平均每节课备课 2-3 小时）
- 批改负担重（一个班 50 人，每周批改数百份作业）
- 个性化教学难（学生水平差异大）
- 家校沟通成本高（每天 1-2 小时）

AI 可以帮你：

- ✅ 备课设计：生成教案、课件大纲、练习题
- ✅ 课件制作：PPT 设计、素材推荐
- ✅ 作业批改：自动批改、错题分析
- ✅ 学情分析：成绩分析、学习报告
- ✅ 家校沟通：通知模板、沟通话术

**实测效果**：一个初中语文老师，用 AI 后备课时间减少 50%，批改效率提升 3 倍。

## 一、备课设计：从 3 小时到 1 小时

### 场景 1：教案生成

**Prompt 模板**：
```
请帮我设计一节 [学科] [年级] 的教案：

课题：[XX]
课时：[XX] 课时
教学目标：[XX]

请输出完整教案，包括：
1. 教学目标（知识与技能、过程与方法、情感态度价值观）
2. 教学重难点
3. 教学准备
4. 教学过程
   - 导入（5 分钟）
   - 新授（20 分钟）
   - 练习（15 分钟）
   - 总结（5 分钟）
5. 板书设计
6. 作业布置
7. 教学反思
```

### 场景 2：课件大纲

**Prompt 模板**：
```
请帮我设计 [课题] 的 PPT 大纲：

课时：[XX] 分钟
学生水平：[XX]

请输出：
1. 每页 PPT 的标题
2. 核心内容（一句话）
3. 建议的视觉元素（图片/图表/视频）
4. 预计时长

总页数：15-20 页
```

### 场景 3：练习题生成

**Prompt 模板**：
```
请帮我出一份 [课题] 的练习题：

题型要求：
- 选择题：[XX] 道
- 填空题：[XX] 道
- 简答题：[XX] 道
- 应用题：[XX] 道

难度分布：
- 基础题：60%
- 中等题：30%
- 拔高题：10%

请输出题目和参考答案。
```

### 场景 4：分层作业设计

**Prompt 模板**：
```
请帮我设计 [课题] 的分层作业：

学生情况：
- A 层（优秀）：[XX] 人
- B 层（中等）：[XX] 人
- C 层（基础）：[XX] 人

请为每层设计作业：
- A 层：拓展性、挑战性题目
- B 层：巩固性、提升性题目
- C 层：基础性、重复性题目

每层作业时长：20-30 分钟
```

## 三、作业批改：从手工到智能

### 场景 1：作文批改

**Prompt 模板**：
```
请帮我批改这篇作文：

学生年级：[XX]
作文题目：[XX]
作文要求：[XX]

学生作文：
[粘贴作文]

请输出：
1. 总体评价（优点、不足）
2. 逐段批注（指出具体问题）
3. 修改建议（3-5 条）
4. 评分（满分 [XX] 分）
5. 鼓励性评语
```

### 场景 2：错题分析

**Prompt 模板**：
```
请帮我分析学生的错题：

题目：[粘贴题目]
学生答案：[粘贴]
正确答案：[粘贴]

请输出：
1. 错误原因分析
2. 知识点漏洞
3. 讲解思路
4. 类似题目推荐（3 道）
```

### 场景 3：作业统计

**Prompt 模板**：
```
请帮我统计作业情况：

作业数据：
[粘贴学生姓名、得分、错题情况]

请输出：
1. 整体情况（平均分、及格率、优秀率）
2. 典型错误分析（错误率最高的 5 题）
3. 需要重点关注的学生
4. 下节课教学建议
```

## 八、常见问题

**Q：用 AI 备课会不会不专业？**
A：不会。AI 生成的是初稿，你需要根据学生实际情况调整。AI 提升效率，不替代专业判断。

**Q：AI 批改作业准确吗？**
A：客观题准确率高，主观题（作文）需要人工复核。建议 AI 批改 + 人工抽查。

**Q：学生知道老师用 AI 会怎么想？**
A：学生会觉得老师与时俱进。关键是教学内容质量，工具只是手段。

**Q：家长会接受吗？**
A：只要教学效果好，家长不关心你用什么工具。AI 帮你节省时间，有更多精力关注学生。

**Q：会影响教师职业吗？**
A：不会。AI 替代的是重复劳动，教师的核心价值（育人、情感交流、价值引导）无法替代。

## 最后一句

AI 不是来替代教师的，是来**替代教师的重复劳动**的。

会用 AI 的教师，把时间花在：
- 关注学生而不是批改作业
- 教学创新而不是重复备课
- 家校沟通而不是机械工作

**记住**：AI 是工具，你是教育者。用对了，效率提升 50%；用错了，教学质量下降。
