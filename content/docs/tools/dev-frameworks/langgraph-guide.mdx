# LangGraph入门指南

LangGraph是LangChain团队推出的Agent框架。

## 核心概念

- State: 状态管理
- Node: 节点
- Edge: 边
- Graph: 图

## 应用场景

- 多轮对话
- 工作流编排
- 复杂Agent
