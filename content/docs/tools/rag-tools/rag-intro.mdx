# RAG技术入门指南

## 什么是RAG

RAG（Retrieval-Augmented Generation）检索增强生成，通过检索外部知识增强LLM的生成能力。

## 工作原理

1. 文档切分
2. 向量化存储
3. 相似度检索
4. 上下文增强
5. 生成回答

## 应用场景

- 企业知识库
- 智能客服
- 文档问答
- 专业领域咨询
