# 构建个人AI助手

## 项目目标

打造一个能理解个人习惯、提供个性化建议的AI助手。

## 技术方案

- 平台：Dify/Coze
- 知识库：个人笔记、日程
- 能力：日程管理、信息检索、建议生成

## 实现步骤

1. 收集个人数据
2. 构建知识库
3. 设计对话流程
4. 持续优化
