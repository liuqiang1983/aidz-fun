# 工作流自动化实战案例

## 案例背景

企业日常运营中有大量重复性工作，通过自动化可以提升效率。

## n8n简介

n8n是开源的工作流自动化工具，支持400+集成。

## 实战案例：自动内容发布

### 场景

每周自动从Notion获取文章，发布到WordPress和社交媒体。

### 工作流设计

1. 定时触发（每周一）
2. 从Notion获取文章
3. AI生成摘要
4. 发布到WordPress
5. 发布到Twitter/微博

### 效果

节省每周2小时手动操作时间。
